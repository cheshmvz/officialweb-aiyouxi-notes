# officialweb-aiyouxi-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存储，不针对任何特定域名或网站，主要用于个人笔记、页面展示或轻量级内容发布。

仓库内容以 HTML 文件为主，附带少量辅助资源（如 CSS、JavaScript、图片等），每个页面通常独立成篇，结构清晰，便于直接浏览和引用。

## 目录说明

```
├── README.md            # 本说明文件
├── index.html           # 入口页面（可选）
├── pages/               # 主要 HTML 页面存放目录
│   ├── example.html
│   └── ...
├── assets/              # 静态资源目录
│   ├── css/
│   ├── js/
│   └── images/
└── docs/                # 文档或说明文件（可选）
```

- `pages/`：存放主要 HTML 页面，每个文件为一个独立页面。
- `assets/`：存放页面引用的样式、脚本、图片等资源文件。
- `docs/`：存放额外的说明文档或笔记，非必需。

## 页面归档说明

每个 HTML 页面均以独立文件形式归档，文件名通常反映页面内容或用途。页面之间无强依赖关系，可单独打开运行。

页面内容可能包含个人笔记、工具页面、演示页面等，不限定具体主题。所有页面均为静态文件，无需后端服务即可正常浏览。

## 维护说明

- 本仓库由个人维护，不定期更新。
- 欢迎提交 Issue 或 Pull Request，但请保持内容简洁，避免引入外部链接或商业内容。
- 如需新增页面，请在 `pages/` 目录下创建新 HTML 文件，并确保资源引用路径正确。
- 不建议在页面中引用外部资源，以保证离线可访问性。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署后在线浏览（需自行配置）。
